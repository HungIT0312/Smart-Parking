# Parking Manage

## Overview:
Smart Parking is a system that helps users find and reserve parking spots intelligently, using technologies such as, IoT, AI, and software...

## Features: 
  - Sign In/ Sign Up
  - View list User
  - View detail vehicle
  - Search

## Technology used
  - React JS
  - API BE (MongoDB)
  - AI Python
  - IOT: sensors
